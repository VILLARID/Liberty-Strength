# 🏋️ Liberty Strength

Liberty Strength is a modern, responsive gym and fitness landing page built with **React** and **Tailwind CSS**. It follows a **mobile-first approach**, ensuring a seamless experience across all devices, from mobile screens to large desktops.

The design focuses on strength, motivation, and a premium fitness experience for athletes and people looking to transform their lifestyle.

---

## ✨ Features

- 📱 Mobile-first responsive design  
- ⚛️ Built with React  
- 🎨 Styled with Tailwind CSS  
- 🧭 Smooth navigation with section-based layout  
- 🏋️ Hero section with strong visual impact  
- 💪 Fitness-focused branding and presentation  
- 📋 Programs showcase  
- 👨‍🏫 Trainers section (in progress)  
- 🚀 Call-to-action for memberships  

---

## 📐 Design Approach

Liberty Strength is developed with a **mobile-first mindset**, meaning:

- Layout optimized for small screens first  
- Scales naturally to tablet and desktop devices  
- Focus on readability and spacing  
- Strong visual hierarchy  
- Modern and premium fitness style
